# 滝行シミュレータ
あなたは修行の身。

滝から流れてくる丸太を避けつつ、煩悩を打ち砕きましょう。

煩悩を打ち砕き続けば、いずれ悟りの境地へ至れるという噂が____

## 操作方法

|アクション | コマンド | 備考 |
| --- | --- | --- |
| 移動 | 左右スワイプ |  |
| 煩悩破壊 | タップ | |
| 無敵化 | 上スワイプ | ゲージ満タン時のみ |

## ルールなど
- 丸太に当たったらゲームオーバー
- ゲージが満タンの時に上スワイプで2秒間無敵
- 煩悩をタップすることでスコアを獲得
- 煩悩が画面外に落ちてしまうとゲージが減る
